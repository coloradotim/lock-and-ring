# Getting Started

1. Choose the microphone you want to use.
2. Sing or speak at rehearsal volume and use Check Mic Setup if you want a quick technical readiness check.
3. Record a short take or import an existing audio take.
4. Review Take Analysis for the current take: signal quality, Lock, Ring, Roughness, Stability, timing, and visual evidence.
5. Let the first take become the initial reference automatically.
6. Make one musical adjustment and record or import another take.
7. Compare the current take against the active reference or keeper. You can also compare against the previous take, another saved take, or an imported file.
8. Adjust the selected region when you want to focus on a chord, vowel, release, phrase section, or the full sung phrase, then update the analysis deliberately.
9. Mark the current selected region as keeper when it is the best-so-far example for that song or section, or record again.

If the app says the signal is too quiet, noisy, clipped, or low confidence, fix the input before trusting the metrics. Move closer, sing a steadier vowel, reduce background noise, or lower the input level if clipping appears.

Use Lock & Ring to compare changes between takes. Do not chase every frame or percentage. Room sound, microphone placement, vowel shape, singer balance, and phrase timing all affect the readings.

The app should describe what changed and how confident it is. It should not make singer-level accusations or combine the whole take into one final score.
