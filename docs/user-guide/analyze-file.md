# Importing a Take

Imported audio is treated as a take. Importing answers: what does this recorded
take show, and should we save, compare, try again, or discard it?

Use Import Take on the Ready screen to load a recording. After import, the app
builds a take from the file and opens Take Analysis using the same scoring path
as microphone-recorded takes.

Imported take review includes the same sections as a recorded take: summary,
confidence, timing / chord behavior, phrase status, and visual evidence.

If the file has clipping, low signal, or noisy sections, trust confidence and
signal warnings more than the raw score.
