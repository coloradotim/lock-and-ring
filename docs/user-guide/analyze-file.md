# Importing a Take

Imported audio is treated as a take. Importing answers: what does this recorded take show, can it serve as a reference, and how does it compare to the current rehearsal take?

Use Import Take on the Ready screen to load a recording. After import, the app builds a take from the file and opens Take Analysis using the same scoring path as microphone-recorded takes.

An imported take may be used as:

- the current take being analyzed
- a comparison reference
- a saved take in the local library
- source material for a keeper region

When an imported take is saved, Lock & Ring copies replayable audio into the local take library. The app does not depend on the original file staying in place.

Imported take review includes the same sections as a recorded take: summary, confidence, whole-take metrics, selected-region metrics, timing / chord behavior, phrase status, and visual evidence.

Imported files should work with the same selected-region flow as recorded takes. The app should auto-detect the likely sung region, let the user adjust the selected region, and allow comparison against the active reference, keeper, previous take, saved take, or another imported file.

If the file has clipping, low signal, or noisy sections, trust confidence and signal warnings more than the raw score.
