# User Guide

Lock & Ring is a take-based rehearsal aid for ensemble sound. Record or import a take, then review Take Analysis for ensemble-level trends in tuning, ring, roughness, stability, timing, visual evidence, and signal confidence.

The app is designed for serious quartets and vocal harmony ensembles working through short rehearsal takes. The usual loop is:

```text
Record/import take → inspect current take → compare to reference → adjust region → listen A/B → mark keeper or record again
```

The first take in a run becomes the initial reference automatically. Later takes compare against the active reference or keeper by default, because the previous take is not always the best comparison target. Users should also be able to compare against the previous take, any saved take, or an imported file.

The app cannot reliably identify which singer is wrong from one microphone.
Treat the output as confidence-aware ensemble feedback, not a verdict.

The app should not collapse a take into one overall score. Use the metric-specific evidence, selected-region analysis, comparison, playback, and your own ears to decide what to keep.
