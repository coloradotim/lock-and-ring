# User Guide

Lock & Ring is a take-based rehearsal aid for ensemble sound. Record or import a take, then review Take Analysis for
ensemble-level trends in tuning, ring, roughness, stability, timing, visual evidence, and signal confidence.

Start here:

- [Getting Started](getting-started.md)
- [Recording a Take](live-rehearsal.md)
- [Take Analysis and Comparison](take-1-take-2.md)
- [Importing a Take](analyze-file.md)
- [Metrics](metrics.md)
- [Signal Quality](signal-quality.md)
- [Reading Visualizations](reading-visualizations.md)

The app cannot reliably identify which singer is wrong from one microphone.
Treat the output as confidence-aware ensemble feedback, not a verdict.
