# User Guide

Lock & Ring is a rehearsal aid for ensemble sound. It listens through one
microphone and reports ensemble-level trends for tuning, ring, roughness,
stability, and signal confidence.

Start here:

- [Getting Started](getting-started.md)
- [Live Rehearsal](live-rehearsal.md)
- [Take 1 / Take 2](take-1-take-2.md)
- [Analyze File](analyze-file.md)
- [Metrics](metrics.md)
- [Signal Quality](signal-quality.md)
- [Reading Visualizations](reading-visualizations.md)

The app cannot reliably identify which singer is wrong from one microphone.
Treat the output as confidence-aware ensemble feedback, not a verdict.
