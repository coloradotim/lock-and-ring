# Live Rehearsal

Live Rehearsal answers: what is happening right now as we sing?

Read the screen from top to bottom:

- Signal Status tells you whether the app trusts the input.
- What Just Happened summarizes recent changes.
- Current Quality gives singer-friendly labels for Lock, Ring, Roughness, and
  Stability.
- Quick Take gives a fast path into Take 1 / Take 2.
- Spectrum / Spectrogram stays secondary because rehearsal feedback matters
  more than raw graphs.

When signal confidence is low, the app avoids strong claims. It may show one
summary such as "Not enough usable signal to evaluate changes yet" instead of
listing repeated metric warnings.

Use Current Quality as a coaching clue. For example, "Mostly aligned" or
"Developing" should guide the next musical experiment, not become a score to
chase.
